# 🌌 Atlas

A minimalist, premium workspace designed for creators, solo founders, and students to organize their files and workflows. Atlas replicates the clean structural design of modern productivity tools while infusing a unique dark-themed aesthetic and refined typography.

## ✨ Features

- **Premium Minimalist UI**: A clean workspace area utilizing a curated dark color palette (`#171717` background, `#D1D5DB` text) for a distraction-free environment to organize files.
- **Efficient File Organization**: Seamlessly manage and structure your ideas, documents, and resources in one centralized hub designed for creators, founders, and students.
- **Collapsible Sidebar Navigation**: A sleek, collapsible sidebar with custom toggle icons and compact navigation items to switch between projects.
- **Integrated AI Workspace**: Smooth AI prompt inputs with dynamic greetings, elegant typography, and flexible model selection to supercharge your productivity.
- **Smooth Animations**: Interactivity powered by Framer Motion, enhancing the user experience with fluid micro-interactions.

## 🚀 Tech Stack

- **Framework**: React 19 + TypeScript
- **Tooling**: Vite
- **Styling**: Tailwind CSS v4
- **Components**: Radix UI Primitives
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🛠️ Getting Started

### Prerequisites

Ensure you have Node.js installed to install the dependencies and run the development server.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Velith15/Atlas-SO.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Atlas-SO
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the Vite development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## 🖌️ Design System

Atlas is meticulously designed with a focus on typography and space:
- **Fonts**: `Inter` for functional UI text, `Baskervville` / `Times New Roman` for elegant serif greetings, and `JetBrains Mono` to establish a minimalist, tech-focused brand identity.
- **Colors**: A deep dark workspace scheme specifically built to reduce visual fatigue, complemented by subtle highlights for interactive elements.
